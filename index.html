Why?

This project aims to keep API developers up to date with the best OpenAPI tooling around. As OpenAPI v3 rolls out, we want to push developers towards that standard and leave OpenAPI v2 behind.

Learn how to convert your specifications to OpenAPI v3, and find v3 tools here to replace outdated v2 tooling, improving your workflow as you go.

Contribute

This project was created by Matthew Trask and Phil Sturgeon with contributions from the community. If you need help, hit Matt or Phil up on twitter.
